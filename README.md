# enum문과 stream lambda를 활용한 계산기

<요구조건>
1. 계산기는 2개의 숫자(실수)를 받을 수 있고 사칙연산을 시행할 수 있다.
2. 계산기는 exit을 입력할 때까지 계속해서 값을 받고 연산 결과를 반환한다.
3. 계산된 결과 값들을 기록하는 컬렉션을 만든다.
4. 컬렉션의 가장 먼저 저장된 데이터를 삭제하는 기능을 만든다.
5. 특정 기준값 입력을 통해 기준값보다 큰 연산결과를 조회할 수 있다.

<실행방법
1. 두개의 실수와 연산자를 차례대로 입력합니다.
   - 올바르지 못한 값을 입력시 재입력을 받습니다.
   - 두번째 실수에 0을 입력하고 나눗셈('/')을 할때 연산을 진행하지 않습니다. 
2. 연산된 결과는 기록됩니다.
3. 기록된 결과를 확인하고 가장 오래된 기록을 삭제할 수 있습니다.
4. 기준 값 입력을 통해 이전의 연산한 결과와 비교하여 기준값보다 큰 연산결과들을 조회할 수 있습니다.
5. 해당 과정을 마무리하고 연산을 계속할 지 여부를 확인합니다.
6. 종료과정에서 exit를 입력시 프로그램을 종료합니다.
